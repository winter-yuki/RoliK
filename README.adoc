= RoliK

ROgueLIke game engine for Kotlin.

== Getting started

.Run example game
[source,bash]
----
$ ./gradlew game:run
----

.Build and run checks and tests
[source,bash]
----
$ ./gradlew build
----

.Generate docs
[source,bash]
----
$ ./gradlew dokkaHtml
----

== Repository organization

* *rolik* -- game engine core
* *rolik-lanterna* -- terminal UI implementation with Lanterna lib
* *game* -- an example game
